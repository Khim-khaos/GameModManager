# Game Mod Manager

Менеджер модов для игр, использующий SteamCMD.

## Описание

Этот проект представляет собой менеджер модов, который позволяет пользователям управлять модами для игр, поддерживающих Steam Workshop. Он использует SteamCMD для загрузки модов и предоставляет пользовательский интерфейс для управления играми и их модами.

## Функциональность

- Добавление, редактирование и удаление игр.
- Указание пути к исполняемому файлу игры, папке с модами и steamcmd.exe.
- Загрузка модов из Steam Workshop через SteamCMD.
- Просмотр установленных модов.
- Запуск игр.
- Встроенный браузер для просмотра Steam Workshop.

## Зависимости

- PyQt5
- PyQtWebEngine
